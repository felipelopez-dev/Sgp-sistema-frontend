Como calcular o valor baixo

  1. 1. No campo "Qual o primeiro valor de carcaça?", insira o segundo dígito do número identificado.

  2. 2. No campo "Qual o segundo valor de carcaça?", insira o terceiro dígito do número identificado.

  3. 3. Verifique se os valores foram digitados corretamente antes de prosseguir.

  4. 4. Clique no botão "Próximo" para continuar o processo.
